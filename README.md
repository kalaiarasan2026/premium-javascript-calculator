# Premium JavaScript Calculator

A modern and responsive calculator built using **HTML**, **CSS**, and **JavaScript**.

## Preview
This project features a premium glassmorphism-style calculator UI with:
- Basic math operations
- Responsive design
- Keyboard support
- Delete and clear buttons
- Expression history display

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Addition, subtraction, multiplication, division, and percentage
- Responsive user interface
- Premium modern design
- Keyboard input support
- Clear all and delete one-by-one
- Expression history
